Documentación de la API REST
==============================
Especificaciones y guías para consumir la API de gestionARTE:
- openapi.yaml         → especificación OpenAPI 3.0 (generada por FastAPI)
- authentication.md    → guía de autenticación JWT
- endpoints.md         → referencia de todos los endpoints con ejemplos
- errors.md            → catálogo de códigos de error y sus significados
FastAPI también genera documentación interactiva automática en /docs y /redoc.
