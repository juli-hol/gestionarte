Archivos Públicos Estáticos
=============================
Recursos que se sirven directamente sin procesar por Vite:
- favicon.ico, logo.svg    → íconos de la aplicación
- robots.txt               → directivas para motores de búsqueda
- manifest.json            → configuración PWA
Los archivos aquí son accesibles en la URL raíz sin rutas adicionales.
