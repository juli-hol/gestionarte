Assets Estáticos Procesados
=============================
Recursos importados y procesados por Vite durante el build:
- images/   → imágenes (.png, .jpg, .svg, .webp)
- fonts/    → fuentes tipográficas personalizadas
- icons/    → iconografía SVG personalizada
A diferencia de /public, estos archivos reciben hash en su nombre para cache-busting.
