Tipos TypeScript Globales
==========================
Definiciones de tipos e interfaces compartidas en todo el frontend:
- api.types.ts       → tipos de respuesta HTTP (ApiResponse<T>, PaginatedResponse<T>)
- common.types.ts    → tipos utilitarios genéricos
- env.d.ts           → tipado de variables de entorno (import.meta.env)
Los tipos específicos de cada feature se definen dentro de su carpeta.
