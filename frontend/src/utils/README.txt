Utilidades y Helpers
======================
Funciones puras de utilidad sin efectos secundarios:
- formatters.ts    → formateo de fechas, moneda, números
- validators.ts    → validaciones de formularios
- constants.ts     → constantes de la aplicación (roles, estados, etc.)
- helpers.ts       → funciones misceláneas reutilizables
No deben importar componentes React ni acceder a la API.
