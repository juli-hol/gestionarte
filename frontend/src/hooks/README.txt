Custom Hooks Globales
======================
Hooks de React reutilizables que no pertenecen a ninguna feature específica:
- useDebounce.ts      → retrasa la ejecución de efectos
- useLocalStorage.ts  → persistencia en localStorage tipada
- useMediaQuery.ts    → responsive design programático
- usePagination.ts    → lógica de paginación reutilizable
- useToast.ts         → sistema de notificaciones
Solo deben contener lógica genérica y transversal.
