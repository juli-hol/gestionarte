Estilos Globales
=================
Estilos CSS/SCSS que aplican a toda la aplicación:
- globals.css       → reset, variables CSS, tipografía base
- variables.css     → design tokens (colores, espaciados, breakpoints)
- animations.css    → keyframes y clases de animación reutilizables
- tailwind.css      → directivas @tailwind si se usa Tailwind CSS
Los estilos de componentes van co-ubicados junto a cada componente.
