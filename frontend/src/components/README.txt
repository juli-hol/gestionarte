Componentes Reutilizables
==========================
Biblioteca de componentes React compartidos entre múltiples features.
Se organiza en tres categorías:
- common/  → componentes de propósito general (Button, Input, Modal, etc.)
- layout/  → estructuras de página (Header, Sidebar, Footer, PageWrapper)
- ui/      → componentes de interfaz visual puro (Badge, Avatar, Spinner)
Ningún componente aquí debe contener lógica de negocio.
