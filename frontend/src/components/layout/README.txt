Componentes de Layout
======================
Estructuras que definen la disposición visual de las páginas:
- AppLayout.tsx     → layout principal autenticado (sidebar + topbar + contenido)
- AuthLayout.tsx    → layout para páginas de login/registro
- Sidebar.tsx       → menú de navegación lateral
- Topbar.tsx        → barra superior con usuario y notificaciones
- PageWrapper.tsx   → contenedor de página con título y breadcrumbs
