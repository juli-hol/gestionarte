Utilidades del Backend
=======================
Funciones helper reutilizables sin efectos secundarios:
- date_utils.py    → manejo y formateo de fechas/timezone
- email_utils.py   → envío de emails transaccionales
- file_utils.py    → manejo de archivos subidos
- pagination.py    → helper de paginación para queries
- validators.py    → validaciones de negocio reutilizables
