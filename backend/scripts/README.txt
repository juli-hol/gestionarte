Scripts de Utilidad del Backend
=================================
Scripts Python ejecutables para tareas administrativas y de mantenimiento:
- seed_db.py          → poblar la BD con datos de prueba/demo
- create_superuser.py → crear usuario administrador inicial
- backup_db.py        → script de backup de la base de datos
- cleanup_old_data.py → limpieza de datos obsoletos
Se ejecutan manualmente o como jobs programados (cron).
