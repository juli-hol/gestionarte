Versiones de Migraciones
=========================
Archivos Python generados por Alembic que describen cambios al esquema.
Cada archivo tiene un ID único, referencia a la revisión anterior (down_revision)
y dos funciones:
- upgrade()   → aplica el cambio (CREATE TABLE, ADD COLUMN, etc.)
- downgrade() → revierte el cambio (DROP TABLE, DROP COLUMN, etc.)
NUNCA editar migraciones ya aplicadas en producción.
